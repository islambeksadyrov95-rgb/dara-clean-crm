## Философия: Data-first минимализм

Дашборд для работы с финансами. Каждый пиксель помогает ЧИТАТЬ и ПРИНИМАТЬ РЕШЕНИЯ.
Референсы: Stripe Dashboard, Adesk.ru, Apple iOS, DashStack UI Kit.

**Ключевое отличие от первой версии:** ВСЁ РЕДАКТИРУЕМОЕ. Плановые значения, цели, параметры роста, лимиты — пользователь может изменять прямо в дашборде, и все зависимые показатели пересчитываются автоматически.

## Правила отображения чисел

- Все суммы: `Intl.NumberFormat('ru-RU')` → `1 234 567 ₸`
- Проценты: одна десятая → `16.6%`
- Отрицательные: красный + минус → `-449 529 ₸`
- Числа: ВСЕГДА по правому краю в таблицах
- **РЕДАКТИРУЕМЫЕ поля:** жёлтый фон `#FFFBEB`, border `1px solid #FDE68A`, при фокусе `border-color: #4880FF`

### Отклонение план/факт

```
Δ ≤ 5%   → зелёный бейдж (в норме)
Δ 5–15%  → жёлтый бейдж (внимание)
Δ > 15%  → красный бейдж (критично)
```

Для расходов — ИНВЕРТИРОВАНО: ниже плана = зелёный.
Всегда дублировать цвет иконкой ↑↓ (colorblind).

```

    ├── ФОТ больничный       284,700 ₸           ░░░░▁░▂▁░░░▃
    └── ФОТ аутсорс          150,000 ₸
  ▶ Материалы              3,917,615 ₸
  ▶ Коммунальные           4,022,877 ₸
  ▶ Поставщик             11,665,442 ₸
▶ ЛОГИСТИКА                8,223,434 ₸  (8.5%)

```
│ Блок         │ Факт 2025 │ ПЛАН 2026 [edit] │ Δ      │ На кв.м. │
│ Производство │ 66.4M     │ [  60.0M       ] │ -9.7%  │ auto     │

## Навигация

- **Hash-routing:** `#overview`, `#finance/2025`, `#finance/2026`, `#finance/calendar`, `#cost`, `#funnel`, `#sales/managers`, `#sales/channels`, `#sales/clients`, `#sales/plan`, `#unit/cac`, `#unit/marketing`, `#unit/growth`, `#goals`
- **Breadcrumb:** `DaraClean > Финансы > 2026 > План/Факт`
- **Переключение периодов:** dropdown сверху (месяц / квартал / год / произвольный)
- **Анимации:** 200ms ease для transitions, 300ms для expand/collapse

## Технические требования

- Чистый JS + Chart.js (как существующий дашборд)
- Новые файлы:
  - `finance-dashboard.js` — финансовый раздел + себестоимость
  - `sales-dashboard.js` — продажи + воронка + менеджеры
  - `unit-economics.js` — CAC, LTV, маркетинг, рост
  - `editable-table.js` — компонент редактируемых таблиц
  - `scenario-engine.js` — пересчёт сценариев при изменении параметров
- Данные: `data/finance-full-data.json` (генерируется ETL)
- Все числа: `Intl.NumberFormat('ru-RU')`
- Все редактируемые значения сохраняются в `localStorage` + экспорт в JSON

## Правила UX

1. **Не больше 5 KPI** на одном экране
2. **Каждая цифра кликабельна** → drill-down
3. **Breadcrumb** сверху
4. **Sticky header** в таблицах
5. **Hover = tooltip** с полной информацией
6. **Анимации** subtle: 200ms ease
7. **Отклонение показывай ВСЕГДА** — не просто "факт", а "факт vs план Δ%"
8. **Редактируемые поля** — жёлтый фон, сразу видно что можно менять
9. **Пересчёт мгновенный** — при изменении input все зависимые поля обновляются без перезагрузки
10. **Все цифры ФАКТИЧЕСКИЕ** — если данных нет, показывать "—", НЕ генерировать фейковые числа
