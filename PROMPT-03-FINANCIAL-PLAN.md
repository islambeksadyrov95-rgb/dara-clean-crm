## ПЛАН ВЫХОДА ИЗ РАЗРЫВА — 3 РЫЧАГА

### Рычаг 1: Ограничение вывода средств

```
Текущий вывод: ~1,395,342 ₸/мес (16.7M / 12)
Решение: Лимит 300,000 ₸/мес до закрытия разрыва

Экономия/мес: 1,395,342 - 300,000 = 1,095,342 ₸
Экономия за 6 мес: 6,572,052 ₸
```

### Рычаг 2: Рост выручки +25%

**А. Поднять средний чек +15%:** 24,612 → 28,304 ₸
- Минимальный заказ = 8 кв.м.
- Экспресс-тариф +30%
- Upsell: антиаллергенная обработка +2,000 ₸
- Пересмотр прайса +10–15%

**Б. Увеличить кол-во заказов +10%:** 342 → 376/мес
- Скорость ответа на лид < 30 мин
- Повторные клиенты: SMS/WhatsApp через 60 дней
- Реферальная программа: скидка 10%

**Эффект:** +25% выручки = +2,105,534 ₸/мес
