## Модель себестоимости услуг DaraClean

### Методология

Используется **Full Absorption Costing** (методология Big4 / МСФО IAS 2) — все расходы, необходимые для оказания услуги, распределяются на единицу продукции (1 кв.м. чистки).

Себестоимость разбита на **6 блоков**. Каждая статья из ДДС отнесена к конкретному блоку или **явно исключена** из себестоимости с обоснованием.

## UNIT-ЭКОНОМИКА: CAC и LTV

### CAC — Cost of Acquisition (по каналам)

**Базис расчёта:** данные продаж за апрель (медианный месяц) экстраполированы на год.

**Апрель — факт:**
- Всего обращений: 596
- Заказов оформлено: 315 (конверсия 52.9%)
- Новых клиентов: 77.6% от обращений
- Маркетинг бюджет апрель: ~1,400,000 ₸ (Google 999K + 2ГИС 373K + прочие 28K)

**CAC по каналам (медиана от апреля):**

| Канал | Бюджет/мес | Обращений | Заказов | Новых заказов* | CAC |
|---|---|---|---|---|---|
| Google Ads | 999,552 ₸ | 183 | 77 | ~69 | **14,486 ₸** |
| 2ГИС | 372,900 ₸ | 158 | 72 | ~61 | **6,113 ₸** |
| Рекомендация | 0 ₸ | 42 | 30 | ~15 | **0 ₸** |
| Постоянный клиент | 0 ₸ | 145 | 123 | 0 | **0 ₸** (повторные) |
| Instagram/TikTok | ~19,500 ₸ | 9 | 4 | ~4 | **4,875 ₸** |

*Новых заказов = заказы × % новых клиентов в канале (Google ~90% новые, 2ГИС ~85%, Рекомендация ~50%)

**Средний CAC (только платные каналы):**
```
CAC платный = (999,552 + 372,900 + 19,500) / (69 + 61 + 4) = 1,391,952 / 134 = 10,388 ₸
```

**Blended CAC (все каналы, включая бесплатные):**
```
Всего новых заказов/мес: ~200 (315 заказов × 63.5% новые клиенты среди оформленных)
CAC blended = 1,391,952 / 200 = 6,960 ₸
```

**Годовой CAC:**
```
Маркетинг бюджет за год: 16,795,722 ₸
Новых клиентов за год: 200 × 12 = ~2,400
CAC годовой blended = 16,795,722 / 2,400 = 6,998 ₸
CAC годовой платный = 16,795,722 / (134 × 12) = 16,795,722 / 1,608 = 10,445 ₸
```

> **ВАЖНО:** Это медианная оценка от данных апреля. Реальный CAC может отличаться в зависимости от сезонности. Рекомендуется собирать данные по продажам аналогичную таблицу каждый месяц для уточнения CAC.

### LTV — Lifetime Value

```
Среднее кол-во заказов за жизненный цикл клиента: 2.5 (экспертная оценка)
Средний чек: 24,612 ₸
LTV = 24,612 × 2.5 = 61,530 ₸

LTV/CAC:
├── Blended:  61,530 / 6,998  = 8.8x ✅ (норма >3x)
├── Платный:  61,530 / 10,445 = 5.9x ✅
├── Google:   61,530 / 14,486 = 4.2x ✅
├── 2ГИС:    61,530 / 6,113  = 10.1x ✅
```

**Payback Period (окупаемость клиента):**
```
Прибыль на заказ: 1,030 ₸
Заказов для окупаемости CAC: 6,998 / 1,030 = 6.8 заказов
При 2.5 заказах/год → окупаемость: ~2.7 года

НО: при маржинальности 4.2% CAC окупается очень долго.
При целевой маржинальности 15%: прибыль на заказ = 3,692 ₸ → окупаемость 1.9 заказов = 9 месяцев ✅
```

### Секция «Break-even» (точка безубыточности)

