# Кэш данных ETL

## `nbrk_usd_kzt.json`

Официальный курс **USD → KZT** (тенге за 1 USD) Национального банка РК по календарным дням.

- Формат: `{ "meta": { "source": "...", "generated": "ISO8601" }, "rates": { "YYYY-MM-DD": number } }`
- Обновление: `python etl/fetch_nbrk_rates.py --from 2025-09-01 --to 2026-03-31`
- При отсутствии даты (выходной) в коде используется **последний известный предыдущий** курс (`fx_usd_kzt.get_rate`).
