# Agbis API — TripOrder (создание выезда)

> Источник: https://doc.minihim.ru/api/double_session/triporder — изменение 29.01.2026
> Раздел: команды с клиентской/пользовательской сессией.

### TripOrder — создание выезда в назначенное время
- Интервалы задаются обязательными `hr` и `hr_to`.
- ⚠ По **клиентской** сессии — не более **5 выездов в день**.
```
GET .../api/?TripOrder={"tp":"1","date":"07.07.2020","hr":"11:00","hr_to":"12:00","address":"...","region_id":"1","car_id":"2","comment":"...","fio":"...","tel":"+7...","mp_status":"1"}&SessionID=
```

**Параметры:**
- `date` — дата резерва (формат `21.12.2020`), **обяз.**
- `hr` — время начала (`"11:30"`), **обяз.** (свободное берётся из `TripsHr`)
- `hr_to` — время окончания (`"11:30"`), **обяз.** (свободное из `TripsHrTo`)
- `car_id` — ID мобильного склада, **обяз.**
- `tel` — телефон клиента, **обяз.**
- `mp_status` — статус выезда, **обяз.**: 0 Новый · 1 Совершён · 2 Отменён · 3 Принял · 4 В пути · 5 Приехал · 6 Пересогласование
- `tp` — тип выезда, необяз.: **1 забрать · 2 доставить (по умолчанию)** ⚠ отличается от `Trips`/`Trip`, где `tp` 0 забрать/1 доставить
- `id` — ID выезда (для редактирования, из `Trips`), необяз.
- `contr_id` — ID клиента, необяз.
- `address` — адрес выезда (обяз. если выезд по клиенту)
- `region_id` — ID региона (из `Regions`), необяз.
- `comment` — необяз.
- `fio` — ФИО клиента, необяз.
- `user_id` — экспедитор, необяз.
- `control_user_id` — диспетчер, необяз.
- `SessionID` — обяз.

**Ответ:** `{ "error":0, "TripID":12345, "Msg":"Зарезервировано" }`
