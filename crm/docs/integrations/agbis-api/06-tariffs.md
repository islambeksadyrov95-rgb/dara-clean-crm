# Agbis API (MiniHim) — Тарификация запросов

> Источник: https://doc.minihim.ru/ru/тарификация_api_запросов — изменение 05.06.2026

## Тариф
> Если количество транзакций за месяц **меньше 2000 шт**, то за первую **1000 запросов и меньше** оплата составляет **3000 руб/месяц**; всё, что выше и по количеству не превышает 1000 запросов — **по 3 рубля за запрос**.

## Что тарифицируется
Тарифицируются **записывающие (коммерческие) команды API** (раздел `05-commercial-session.md`):
`ContragForAll`, `AddBonusForAll`, `SaveOrderForAll`, `UpdateOrderForAll`,
`ChangeStatusOrdersForAll`, `SetOrderImagesForAll`, `PayForAll`, `ReturnPayForAll`, `TripOrder`.

Статистику выполненных коммерческих команд можно смотреть командой **`ExecutedApiCount`** (по дням).

## Вывод для интеграции
- **Чтение справочников и синхронизация** (`PriceList`, `ContrTree`, `OrderByDateTimeForAll`, `ClientsByDateTimeForAll`, `LastChangeOrder` и т.п.) — по этой формулировке тарификация относится к коммерческим (записывающим) командам; читающие команды без сессии и с пользовательской сессией в тариф «3 руб/запрос» по тексту не входят. **Уточнить у Леонида**, считаются ли читающие команды в лимит 1000/2000.
- При активной записи заказов держать в уме: до 1000 коммерческих запросов/мес — фикс 3000 руб; сверх — 3 руб/шт.
