# Requirements: Dara Clean CRM

**Defined:** 2026-05-14
**Core Value:** Менеджер открывает систему, видит кому звонить, за 2 клика создаёт заказ или отправляет WhatsApp

## v1 Requirements

### Data Import

- [ ] **IMP-01**: Администратор загружает Excel файл (База Агбис.xlsx) через веб-форму
- [ ] **IMP-02**: Система нормализует телефоны в формат E.164 (+7XXXXXXXXXX) из 7 форматов
- [ ] **IMP-03**: Система дедуплицирует клиентов по номеру телефона при импорте
- [ ] **IMP-04**: Система подсчитывает количество заказов, общую сумму и средний чек на клиента
- [ ] **IMP-05**: Система присваивает RFM-сегмент: Новый (1 заказ), Повторный (2-3), Постоянный (4+), В риске (>90 дней), Потерянный (>180 дней)
- [ ] **IMP-06**: Администратор видит результат импорта: сколько клиентов создано/обновлено/пропущено

### Client Management

- [ ] **CLT-01**: Менеджер видит таблицу всех клиентов с поиском и фильтрами по сегменту
- [ ] **CLT-02**: Менеджер открывает карточку клиента с историей заказов и звонков
- [ ] **CLT-03**: Таблица отображает: имя, телефон, сегмент, заказов, потрачено, последний заказ, дней без заказа

### Call Queue (CRM)

- [ ] **CRM-01**: Менеджер видит очередь клиентов чей последний заказ был N дней назад (настраиваемый фильтр)
- [ ] **CRM-02**: Очередь отсортирована по давности последнего заказа (старые первые)
- [ ] **CRM-03**: Система блокирует клиента за менеджером при начале звонка (locked_by + 10 мин expiry)
- [ ] **CRM-04**: Менеджер нажимает "Дозвонился" и видит форму создания заказа
- [ ] **CRM-05**: Менеджер нажимает "Не дозвонился" и видит сгенерированное WhatsApp сообщение
- [ ] **CRM-06**: Статистика дня видна менеджеру: звонков сделано, успешных, заказов создано

### Order Creation

- [ ] **ORD-01**: Менеджер создаёт заказ из карточки звонка: выбирает услуги (ковры/шторы/мебель/клининг)
- [ ] **ORD-02**: Система автоматически рассчитывает скидку по сетке: 5% повторный, 10% >30К, 15% комплекс
- [ ] **ORD-03**: Менеджер добавляет стоимость и комментарий к заказу
- [ ] **ORD-04**: Заказ сохраняется с привязкой к клиенту и менеджеру

### WhatsApp Integration

- [ ] **WA-01**: Система генерирует персонализированное сообщение через OpenRouter (имя клиента, дата заказа, скидка)
- [ ] **WA-02**: Менеджер видит сгенерированное сообщение и может его скопировать
- [ ] **WA-03**: Кнопка "Открыть WhatsApp" формирует wa.me deep link с текстом сообщения
- [ ] **WA-04**: Шаблон-fallback работает без API ключа OpenRouter

### Authentication & Access

- [ ] **AUTH-01**: Пользователь входит по email/паролю через Supabase Auth
- [ ] **AUTH-02**: Две роли: менеджер (звонки, заказы) и админ (импорт, настройки, все данные)
- [ ] **AUTH-03**: Менеджер видит только свои звонки и заказы, админ видит всё

### Infrastructure

- [ ] **INF-01**: Приложение задеплоено на Vercel и доступно по URL
- [ ] **INF-02**: База данных в Supabase (PostgreSQL) с RLS политиками
- [ ] **INF-03**: Realtime обновление очереди звонков через Supabase Realtime

## v2 Requirements

### Motivation & KPI

- **MOT-01**: KPI калькулятор для менеджеров (конверсия, повторные, средний чек, кросс-продажи)
- **MOT-02**: KPI калькулятор для курьеров (адреса/день, попутные заборы, допродажи)
- **MOT-03**: KPI калькулятор для цеха (нормативы, рекламации, время обработки)
- **MOT-04**: Бонусные пороги с привязкой к плану

### Financial Calculator

- **FIN-01**: Калькулятор эффекта скидок на план 2026
- **FIN-02**: Моделирование: доп.выручка vs стоимость скидок vs чистый эффект
- **FIN-03**: Точка безубыточности при разных % повторных

### Analytics

- **ANL-01**: Дашборд: repeat rate, средний чек по сегментам, динамика по месяцам
- **ANL-02**: Когортный анализ клиентов по месяцу первого заказа
- **ANL-03**: Рейтинг менеджеров по конверсии и выручке

### Advanced CRM

- **ADV-01**: Планирование обратного звонка (call-back scheduling)
- **ADV-02**: Сегментные кампании (массовый обзвон по сегменту)
- **ADV-03**: Интеграция с Агбис API для автоматического импорта

## Out of Scope

| Feature | Reason |
|---------|--------|
| Автоматическая отправка WhatsApp | Требует одобрения Meta Business API, сложный процесс |
| GPS трекинг курьеров | Нет инфраструктуры, требует мобильного приложения |
| Мобильное приложение | Менеджеры работают на компе |
| Email рассылки | WhatsApp основной канал в KZ |
| Scheduling/dispatch | Другая система (Агбис) |
| Интеграция с Kaspi | Нет открытого API |
