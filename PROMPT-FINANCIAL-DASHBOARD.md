## Задача

Добавить в существующий дашборд (`D:/Mind map/Dara Clean/dashboard/`) новую систему финансового планирования, себестоимости, unit-экономики и продаж. Всё — с редактируемыми планами и автоматическим пересчётом.

## Порядок реализации

```
1. ETL: extract_finance_full.py → finance-full-data.json    (PROMPT-01)
2. Себестоимость: модель + визуализация                      (PROMPT-02 + 05)
3. Финансы 2025 факт: tree-table + waterfall + heatmap       (PROMPT-03 + 05)
4. Финансы 2026 план: editable table + сценарии              (PROMPT-03 + 05)
5. Воронка: SVG пирамида + drill-down                        (PROMPT-04 + 05)
6. Продажи: менеджеры + каналы + прогресс-трекер             (PROMPT-04 + 05)
7. Unit-экономика: CAC/LTV таблицы + рост 2026–2028          (PROMPT-02 + 05)
8. Цели (OKR): editable цели + прогресс                      (PROMPT-03 + 05)
```

## Ключевые принципы

- **ВСЁ РЕДАКТИРУЕМОЕ:** планы, цели, параметры роста — изменяются в дашборде, зависимые показатели пересчитываются мгновенно
- **ВСЕ ЦИФРЫ ФАКТИЧЕСКИЕ:** данные только из Excel. Если нет — показывать "—"
- **Себестоимость = 6 блоков:** Производство (68.6%), Логистика (8.5%), Маркетинг (17.3%), Продажи (0.7%), Налоги (3.2%), Операционные (1.7%)
- **Кассовый разрыв — приоритет #1:** дашборд показывает текущий дефицит и прогноз выхода
- **Средний чек: 24,612 ₸** (из медианы апреля, ~315 заказов/мес)
- **CAC blended: 6,998 ₸**, CAC платный: 10,445 ₸
- **LTV: 61,530 ₸** (2.5 заказа за жизненный цикл)
- **Валовая маржа: 4.2%** — критически низкая (норма 25–35%)
